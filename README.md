 📊 Amazon Sales Dashboard - Power BI

 📌 Overview
This is my first Power BI dashboard project. It provides an interactive analysis of Amazon sales data to help understand sales performance, customer trends, and business insights through visualizations.

 🎯 Features
- Sales Overview
- Revenue Analysis
- Product Performance
- Regional Sales Analysis
- Interactive Filters (Slicers)
- KPI Cards
- Charts and Visualizations

 🛠️ Tools Used
- Microsoft Power BI
- Power Query
- DAX

 📂 Files
- `Amazon dashboard.pbix` - Power BI report file

 📈 Learning Outcomes
This project helped me learn:
- Data Cleaning using Power Query
- Data Modeling
- DAX Measures
- Dashboard Design
- Data Visualization Best Practices
